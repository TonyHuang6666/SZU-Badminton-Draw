# 使用说明

## 1. 准备名单

打开软件后点击“生成名单模板”，得到 `深大羽协参赛名单模板.xlsx`。

模板列说明：

| 列名 | 用途 |
| --- | --- |
| 姓名 | 单打选手姓名，或双打第一名选手姓名 |
| 搭档 | 双打搭档姓名 |
| 队伍 | 团体赛队伍名 |
| 是否种子 | 填“是”表示种子，不是种子可以留空或填“否” |
| 种子序号 | 可选，例如 1、2、3、4 |
| 备注 | 可选，导出时会保留 |

单打只需要填写“姓名”。双打填写“姓名”和“搭档”。团体赛优先读取“队伍”，如果“队伍”为空则读取“姓名”。

## 2. 设置抽签

软件中需要设置：

- 比赛模式：单项赛淘汰赛、单项赛循环赛、团体赛淘汰赛、团体赛循环赛。
- 项目类型：单打、双打、团体。
- 小组数：要分成多少个组。
- 算法版本：推荐使用“公开规则算法 v2”。
- 随机种子：可以现场生成，也可以由裁判长、双方代表或公开方式共同确定。

## 3. 预览和导出

点击“预览抽签”后，右侧会显示：

- 总分组
- 第一轮名单
- 轮空或直接晋级名单

确认无误后点击“导出结果 Excel”。

## 4. 复现结果

若需要复查抽签结果，保留以下信息：

- 原始名单 Excel
- 比赛设置
- 算法版本
- 随机种子
- 输出文件中的输入哈希

任何人用同样输入重新生成，都应得到相同结果。
